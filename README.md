# Analog-Interface-for-ADC-input

This project presents the **design, simulation, and practical implementation** of an analog signal conditioning interface intended for preparing sensor signals for **analog-to-digital conversion (ADC)**.

The system processes a sensor signal through amplification, filtering, programmable gain control, and rectification to produce a properly conditioned waveform suitable for ADC input.

---

## Overview

Many sensors generate signals that are **low amplitude**, **noisy**, or **bipolar**, which makes them unsuitable for direct connection to an ADC.  
This project implements a complete **analog front-end (AFE)** that conditions the signal before digitization.

The system performs:

- Differential signal amplification
- Noise filtering
- Programmable gain adjustment
- Full-wave rectification

---

## System Architecture

The analog signal chain is structured as follows:

---

### Sensor Interface
The input signal is generated by a sensor and may have **low amplitude and differential characteristics**, requiring amplification and conditioning before further processing.

### Instrumentation Amplifier
An instrumentation amplifier stage is used to:

- Amplify low-level differential signals from sensor
- Provide **high input impedance**  
- Achieve **high common-mode rejection**

### Analog Filter
A filtering stage removes unwanted noise and limits the signal bandwidth, improving signal quality before further processing.

### Programmable Gain Amplifier (PGA)
The **PGA stage** allows adjustment of the signal gain depending on the input amplitude, ensuring efficient utilization of the ADC input range.

### Full-Wave Rectifier
A precision **full-wave rectifier** converts the bipolar signal into a waveform containing only **positive alternances**, enabling compatibility with ADCs that cannot process negative voltages.

## Practical Implementation

In addition to circuit design and simulations, the system was **implemented and tested in hardware**.

The practical implementation includes:

- Assembly of the analog front-end circuit
- Measurement and verification of the signal conditioning stages
- Experimental validation of amplification, filtering, and rectification stages
- Comparison between **simulation results and measured signals**

This implementation demonstrates the real-world operation of the designed analog signal conditioning interface.
